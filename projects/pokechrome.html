PokéChrome

PokéChrome is a Chrome Extension I created during DVHACKS 2019. This fun Pokémon collector also features a Pokémon market where you can trade with other users, including your friends! You can also view your other friend's party. Go Collect Them All!

This project was made in collaboration with Allen Chang. Github: Sylverost. Website: sylverost.com

Here you can see how the Pokémon appear on your screen. You can set it to spawn periodically or manually spawn one yourself (this feature basically makes it so you can cheat but whatever...).

This is the dashboard where you can see your cash. We also made an account system but passwords are currently unhashed so please don't use your real password.

Install the extension by going to Chrome Extensions page - > Turn on Developer Mode - > Download the extension here: PokéChrome - > Unzip the zip file or clone the repository - > On your extensions page press "Load Unpacked" and select the folder - > Click on the extension on the top right hand corner of the screen and login.
